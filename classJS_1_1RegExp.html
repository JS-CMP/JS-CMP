Detailed Description

Represents a JavaScript-like RegExp object in C++.

The RegExp class is a representation of a JavaScript regular expression object. It inherits from JS::InternalObject and provides methods to manipulate and interact with regular expressions.

https://262.ecma-international.org/5.1/#sec-15.10
