Detailed Description

Represents the string object in C++.

The String class represents a JavaScript-like string object in C++. It provides the different methods defined in the ECMAScript specification for the String object. It also provides a method to get an internal object that holds the methods of the property prototype.

https://262.ecma-international.org/5.1/#sec-15.5
