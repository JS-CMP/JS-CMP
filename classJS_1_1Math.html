Detailed Description

Provides mathematical functions similar to JavaScript's Math object.

The Math class offers a set of static methods that mimic the behavior of the JavaScript Math object. Each function accepts a vector of JS::Any arguments to simulate JavaScript's dynamic typing. The methods implement mathematical operations such as trigonometric functions, exponentiation, logarithm, rounding, and random number generation.

const JS::Any &  args 
)
static

Returns an approximation of the arctangent of the quotient y/x.

The first argument is interpreted as y and the second as x. The signs of y and x are used to determine the quadrant of the result. The returned value is in radians and lies within the range [-π, +π].
