Detailed Description

Represents a JavaScript-like variant type in C++ that can hold multiple types of values.

The Any class can hold values of various types, similar to JavaScript's dynamic typing. It provides overloaded operators for arithmetic and comparison, making it easy to work with in a JavaScript-like syntax.
