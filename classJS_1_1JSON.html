Detailed Description

Provides JSON parsing and stringification similar to JavaScript's JSON object.

The JSON class offers static methods that mimic the behavior of the JavaScript JSON object. It provides parse and stringify methods as specified in ECMA-262 5.1 Section 15.12.

The JSON object:

  • Has [[Prototype]] set to the standard built-in Object prototype
  • Has [[Class]] set to "JSON"
  • Has [[Extensible]] set to true
  • Does not have [[Construct]] (cannot be used with new)
  • Does not have [[Call]] (cannot be invoked as a function)

Parses a JSON text and produces an ECMAScript value.

The parse function parses a JSON text (a JSON-formatted String) and produces an ECMAScript value. The optional reviver parameter is a function that takes two parameters (key and value). It can filter and transform the results.

Returns a String in JSON format representing an ECMAScript value.

The stringify function returns a String in JSON format representing an ECMAScript value. It can take three parameters:

  • value: An ECMAScript value, usually an object or array (can also be String, Boolean, Number, or null)
  • replacer (optional): A function that alters stringification, or an array as a whitelist
  • space (optional): A String or Number for whitespace injection
Parameters
thisArgIgnored for this method.
argsA vector containing the value and optional replacer and space parameters.
Returns
A JS::Any containing the JSON string, or undefined if the value cannot be stringified.
Exceptions
TypeErrorif the structure is cyclical.
