Detailed Description

Represents the date object in C++.

The Date class represents a JavaScript-like date object in C++. It provides the different methods defined in the ECMAScript specification for the Date object. It also provides a method to get an internal object that holds the methods of the property prototype.

https://262.ecma-international.org/5.1/#sec-15.9
